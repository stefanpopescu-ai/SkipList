Skip List — Algoritm

Un Skip List este o structură de date probabilistică ce permite timp mediu O(log n) pentru căutare, inserare și ștergere — similar cu un BST echilibrat, dar mai simplu de implementat.

Structură: Mai multe niveluri de liste înlănțuite. Nivelul 0 este lista completă sortată. Nivelurile superioare conțin mai puține elemente (benzi rapide).

Inserare: Se inserează la nivelul 0, apoi se aruncă o monedă pentru promovare. Promovarea continuă până la pajură sau până la crearea unui nivel nou (cel mult un nivel nou per inserare).

Căutare: Se pornește de la nivelul superior și se merge la dreapta; se coboară un nivel când valoarea următoare este prea mare. O(log n) așteptat.

Ștergere: Se găsește nodul pe toate nivelurile și se refac pointerii înainte.
